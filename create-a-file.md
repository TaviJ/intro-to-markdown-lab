# How to Create a File Using the Terminal

Below are several commands that can be used to create files from the terminal depending on the type of workflow you prefer.

## 1. Using **touch** Command
 The _touch_ command creates an empty file.

 **Example**
 ```touch index.html
touch app.js
touch style.css
```

## 2. Using **echo** Command
 The _echo_ command creates an empty file and puts text inside it immediately.

 **Example**
 `echo "Hello world!" > greeting.txt`

 ## 3. Using **cat** Command 
  The _cat_ Command can create a file and let you type into it right away.

  **Example**
  'cat > notes.txt'
  > type you text, than press: _CTRL + D_
